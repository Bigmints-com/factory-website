# Project Xtara - Root Repository

This is the root repository for the Project Xtara ecosystem, containing shared assets, documentation, and configuration files.

## 🏗️ Project Structure

This workspace contains four main projects under the `project-xtara` umbrella:

```
project-xtara/
├── assets/                    # Shared assets and resources
├── career/                    # Flutter mobile app (separate repository)
├── career-ai/                 # Python AI/ML scripts and services (separate repository)
├── xtara/                     # Next.js website (separate repository)
├── xtara-firebase/            # Firebase operations (separate repository)
└── [shared files]             # Configuration and documentation
```

## 📱 Individual Projects

Each project is maintained as a separate repository:

### Career (Flutter App)
- **Repository**: Separate Flutter repository
- **Tech Stack**: Flutter, Dart, Firebase, GetX
- **Purpose**: Mobile application for career guidance

### Career-AI (Python AI/ML)
- **Repository**: Separate Python repository
- **Tech Stack**: Python, Google Cloud, Vertex AI, Firebase
- **Purpose**: AI/ML pipeline for career recommendations

### Xtara (Website)
- **Repository**: Separate Next.js repository
- **Tech Stack**: Next.js, React, TypeScript, Tailwind CSS
- **Purpose**: Marketing website and landing pages

### Xtara-Firebase (Firebase Operations)
- **Repository**: Separate Firebase repository
- **Tech Stack**: Firebase, Cloud Functions, Firestore
- **Purpose**: Backend services and database operations

## 🎨 Shared Assets

This repository contains:
- **Shared assets**: Images, icons, and design resources
- **Configuration files**: Common configuration templates
- **Documentation**: Project-wide documentation and specifications
- **Data files**: Shared data resources and schemas
